# Metric walkthrough notebooks

These notebooks use the **Facebook** ground-truth graph from the EPGM repository as a tactile teaching example.

Files:
- `facebook_gcc_walkthrough.ipynb`
- `facebook_alcc_walkthrough.ipynb`
- `facebook_k4_density_walkthrough.ipynb`
- `facebook_c3_walkthrough.ipynb`
- `facebook_gcd11_walkthrough.ipynb`

Run them from within the `cluster_analysis/` directory or from `cluster_analysis/notebooks/`.

Notes:
- The notebooks import from `../metrics.py` when needed.
- `facebook_gcd11_walkthrough.ipynb` requires **ORCA** on your `PATH`.
- The notebooks are intentionally pedagogical rather than optimized.
